Problem Set

Open the console and try following functions...

  • printReverse() - a function that takes an array as an argument and prints out its elements in reverse order.
  • isUniform() - a function that takes an array as an argument and returns true if all elements are identical in the array.
  • sumArray() - a function that takes an array as an argument and returns sum of elements in the array.
  • max() - a function that takes an array as an arguments and returns its maximum value in the array.
